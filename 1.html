17. 서버와 클라이언트

- web browser 와 web server 가 깔려있는 두개의 컴퓨터가 있다. Web server 컴퓨터는 http://info.cern.ch 라는 주소를 가지고 있고, index.html이라는 파일이 있다. Web browser 의 주소창에 http://info.cern.ch/index.html 입력하면 web server는 index.html의 정보를 전기적 신호로 보낸다. 이는 web browser는 client , web server 는 server의 역할을 한다고 볼 수 있다.
